# Frontend of Pogclans
> The frontend of [pogclans.com](https://pogclans.com) WIP

## Installation
`
cd pogclans` <br />`
npm i
`

## Running
Running the build version <br />
`
npm run build
`
<br>
Running the dev version <br />
`
npm run dev
`
